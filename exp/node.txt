查看数据表的编码：
 show create table Student;

修改表的编码方式：
 alter table Student default character set utf8;
该命令用于将表Student的编码方式改为utf8；

修改字段的编码方式：
 alter table Student change Sname Sname varchar(20) character set utf8;
该命令用于将表Student中Sname字段的编码方式改为utf8
